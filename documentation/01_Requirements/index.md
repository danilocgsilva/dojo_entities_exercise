# Requirements

Create a web application with the following features:

## General features

* An home page to present the system and hide informations from users that is not logged.

## User workflow

* A login page
* Logout

## Person feafures

* Register person
* Search person
* CRUD for peoples

## Housing features

* Register housings
* Search housing
* CRUD for housings

## API

* CRUD api for housings and peoples
